Use Twitter to receive noticies of comments when they are left on your WordPress site.

== Description ==

wpCommentTwit is a plugin that will notify you of a new comment on your blog by DirectMessage (DM) via Twitter.

You will only receive a DM if the comment is not flagged as spam or was approved manually.

Currently this plugin utilizes the bit.ly hash format for shortening URLs.

Using wpCommentTwit is fairly simple and has two basic requirements:

  1. You have a twitter account (you send yourself Direct Messages).
  2. Bit.ly account - http://bit.ly - Username and API Key

== Installation ==

Installation is easy and can be done by following the below simple instructions.

1. Upload `wpCommentTwit` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter the required information
